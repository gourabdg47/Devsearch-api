DevSearch API Documentation

Project Overview

DevSearch API is a powerful search engine API service that provides structured data from web searches. It offers customizable search verticals, AI-powered features, and developer-friendly tools.

Key Features

  • Basic search results in JSON format
  • Customizable search verticals
  • AI-powered content summarization
  • Multi-lingual support
  • Real-time trending topics
  • Developer-friendly features (SDKs, documentation)

Key Components

1. Main Application (api/main.py)

The entry point of the application, setting up FastAPI and including routers.

Reference: devsearch-api/api/main.py

2. Configuration (api/config.py)

Manages application settings using Pydantic BaseSettings.

Reference: devsearch-api/api/config.py

3. Database (api/db/database.py)

Sets up the database connection using SQLAlchemy.

Reference: devsearch-api/api/db/database.py

4. Models

Define data structures for the application.

  • Custom Vertical Model (api/models/custom_vertical.py)
  • Search Result Model (api/models/search_result.py)

5. Routers

Handle API endpoints for different features.

  • Search Router (api/routers/search.py)
  • Custom Verticals Router (api/routers/custom_verticals.py)
  • AI Features Router (api/routers/ai_features.py)

6. Services

Implement core business logic.

  • Search Engine Service (api/services/search_engine.py)
  • Custom Vertical Service (api/services/custom_vertical_service.py)
  • AI Service (api/services/ai_service.py)

7. Dependencies (api/dependencies.py)

Manages dependency injection for the application.

Reference: devsearch-api/api/dependencies.py

8. Database Migrations (alembic/)

Handles database schema migrations using Alembic.

Reference: devsearch-api/alembic/env.py

Setup and Running

  1. Install dependencies: pip install -r requirements.txt
  2. Set up the database and run migrations: alembic upgrade head
  3. Run the application: uvicorn api.main:app --reload

API Endpoints

  • GET /api/v1/search/: Perform a search
  • POST /api/v1/custom/: Create a custom vertical
  • GET /api/v1/custom/: List custom verticals
  • GET /api/v1/custom/{vertical_id}: Get a specific custom vertical
  • GET /api/v1/ai/summarize: Summarize text using AI

Testing
